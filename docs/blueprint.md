# **App Name**: Evergreen Notes

## Core Features:

- Note Creation: Create new notes with a title and body text.
- Note Editing: Edit the title and body text of existing notes.
- Note Deletion: Delete unwanted notes.
- Note Browsing: Browse a list of all notes, sorted by most recent.
- Progress Tracker: Track user's streak and progress, including decks and topics.

## Style Guidelines:

- Primary color: Lightish brown (#A67B5B) to give a warm and earthy feel.
- Background color: Light green (#F1F8E9), almost white, for a clean, distraction-free writing space.
- Accent color: Dark teal (#008080), to provide contrast on UI elements such as save or edit buttons.
- Splash of gold (#FFD700) for highlights and important elements.
- Body and headline font: 'Inter', a modern sans-serif known for its readability.
- Sidebar font: A playful, bubble-style font to create a calm and welcoming atmosphere.
- Simple, outlined icons for actions like edit, delete, and save.
- Clean, minimalist layout with clear separation of title and body text.
- In the sidebar, display options within speech bubbles.
- Display motivational lines of encouragement in gold writing.
- Subtle fade-in animations for newly created notes and smooth transitions when switching between notes.
